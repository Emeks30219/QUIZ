# 💻 Computer Quiz

A simple command-line quiz game built in Python that tests your knowledge of common computer science acronyms.

---

## 📋 Description

This program presents the user with 7 questions about well-known computing terms. It tracks the player's score and displays a final percentage at the end of the quiz.

---

## 🚀 How to Run

Make sure you have Python installed (Python 3 recommended), then run:

```bash
python computer_quiz.py
```

---

## 🎮 How to Play

1. When prompted, type `yes` to start the quiz (any other input will exit the program).
2. Answer each question by typing the full expanded form of the acronym.
3. Answers are **not case-sensitive** — `RAM`, `ram`, and `Ram` are all accepted.
4. Your score and percentage will be displayed at the end.

---

## ❓ Questions Covered

| # | Acronym | Answer |
|---|---------|--------|
| 1 | RAM | Random Access Memory |
| 2 | CPU | Central Processing Unit |
| 3 | USB | Universal Serial Bus |
| 4 | HTML | Hyper Text Markup Language |
| 5 | GPU | Graphics Processing Unit |
| 6 | ROM | Read Only Memory |
| 7 | PSU | Power Supply Unit |
